⚽ 足球赛事管理系统 ⚽

一个基于 Python  和 MySQL 的简易足球信息管理系统

✨ 核心功能 & 技术亮点

首页概览: 集中查看所有球队和赛事的基础信息。


添加球员 (触发器): 新增球员时，由数据库触发器自动校验球员年龄，确保其符合规定（例如大于16岁） 。


删除球员 (事务): 根据球员姓名删除记录，操作被封装在事务中，确保球员主表及其位置子表的数据被原子性地、安全地移除，从而保证数据一致性 。


球员转会 (存储过程): 调用数据库存储过程来处理球员转会，只需输入球员ID和新球队ID即可完成后台的数据校验与更新 。


查询俱乐部球员 (视图): 基于预先定义的数据库视图 (view_club_player_details) ，按俱乐部名称即可查询该队所有球员的详细信息。
